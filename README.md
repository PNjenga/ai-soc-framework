# AI-Driven Security Operations Center (SOC) Framework

**Capstone Project**

An AI-powered SOC framework using XGBoost, LSTM, SHAP/LIME, MITRE ATT&CK, and Human-in-the-Loop Governance.

## Project Structure
- `docs/` — Documentation and full report
- `src/` — Prototype source code (Layered Architecture)
- `images/` — Design diagrams
- `models/` — Trained models
- `appendices/` — Supporting materials

## Key Features
- Four-layer architecture
- Explainable AI (XAI)
- Human-in-the-loop governance

See `docs/AI-Driven-SOC-Framework-Report.md` for full details.
